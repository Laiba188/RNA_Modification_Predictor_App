# AI-Based RNA Modification Prediction Web App

This web application predicts RNA modification sites including:

- m6A (N6-Methyladenosine)
- m5C (5-Methylcytosine)
- m1A (N1-Methyladenosine)
- Ψ (Pseudouridine)

## Features

- RNA sequence input
- FASTA file upload
- Modification prediction
- Probability scores
- Highlighted sequence visualization
- CSV result download
- AI-based novel modification detection
